KONVERTREST API

Unit Conversion Engine — Length · Weight · Temperature

API Reference
POST /convert/length
Convert between mm, cm, m, km, in, ft, yd, mi. 
{
  "value": 100,
  "from_unit": "cm",
  "to_unit": "in"
}
POST /convert/weight
Convert between mg, g, kg, t, oz, lb, st. 
{
  "value": 70,
  "from_unit": "kg",
  "to_unit": "lb"
}
POST /convert/temperature
Convert between C, F, K. 
{
  "value": 100,
  "from_unit": "C",
  "to_unit": "F"
}
GET /units
Returns all supported units per category.
GET /health
Health check endpoint. Returns {"status":"ok"}.
